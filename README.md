# Newborn Timer

Start

- 1 minute

  - APGAR (Voice)

- 5 minutes

  - 5 minute APGAR

- 10 minute

  - 10 minute APGAR
  - Check Temp
  - Check Blood pressure

- 15 minutes

  - 15 minute APGAR

- 30 minutes

  - Check glucose and draw labs
  - Give vitamin K
  - Give Erythromycin
  - Start IV fluids

- 60 minutes

  - Check glucose if fluids not running
  - Check Temp
  - Check Blood pressure
  - Give IV medications
  - Start IV fluids if not started at 30 minutes
